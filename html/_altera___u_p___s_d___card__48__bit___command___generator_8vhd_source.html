00002 -- This module takes a command ID and data, and generates a 48-bit message for it.
00003 -- It will first check if the command is a valid 48-bit command and produce the
00004 -- following outputs:
00005 -- 1. o_dataout ->      a single bit output that produces the message to be sent to the 
00006 --                                      SD card one bit at a time. Every time the i_message_bit_out input
00007 --                                      is high and the i_clock has a positive edge, a new bit is produced.
00008 -- 2. o_message_done -> a signal that is asserted high when the entire message has been
00009 --                                              produced through the o_dataout output.
00010 -- 3. o_valid   ->      is a signal that is asserted high if the specified message is valid.
00011 -- 4. o_response_type -> indicates the command response type.
00012 -- 5. o_returning_ocr -> the response from the SD card will contain the OCR register
00013 -- 6. o_returning_cid -> the response from the SD card will contain the CID register
00014 -- 7. o_returning_rca -> the response from the SD card will contain the RCA register
00015 -- 8. o_returning_csd -> the response from the SD card will contain the CSD register
00016 -- 9. o_data_read -> asserted when the command being sent is a data read command.
00017 -- 10. o_data_write -> asserted when the command being sent is a data write command.
00018 -- 11. o_wait_cmd_busy ->       is set high when the response to this command will be
00019 --                                                      followed by a busy signal.
