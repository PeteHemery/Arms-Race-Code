00002 -- This module is a clock generator for the SD card interface. It takes a 50 MHz
00003 -- clock as input and produces a clock signal that depends on the mode in which the
00004 -- SD card interface is in. For a card identification mode a clock with a frequency of
00005 -- 390.625 kHz is generated. For the data transfer mode, a clock with a frequency of
00006 -- 12.5MHz is generated. 
00007 --
00008 -- In addition, the generator produces a clock_mode value that identifies the frequency
00009 -- of the o_SD_clock that is currently being generated.
