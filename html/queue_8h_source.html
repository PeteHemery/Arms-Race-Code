01180 /*
01181  * xQueueAltGenericSend() is an alternative version of xQueueGenericSend().
01182  * Likewise xQueueAltGenericReceive() is an alternative version of
01183  * xQueueGenericReceive().
01184  *
01185  * The source code that implements the alternative (Alt) API is much
01186  * simpler      because it executes everything from within a critical section.
01187  * This is      the approach taken by many other RTOSes, but FreeRTOS.org has the
01188  * preferred fully featured API too.  The fully featured API has more
01189  * complex      code that takes longer to execute, but makes much less use of
01190  * critical sections.  Therefore the alternative API sacrifices interrupt
01191  * responsiveness to gain execution speed, whereas the fully featured API
01192  * sacrifices execution speed to ensure better interrupt responsiveness.
01193  */
