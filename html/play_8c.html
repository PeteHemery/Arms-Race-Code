void vPlayProgram ( portCHAR *  pcFileName,
portBASE_TYPE  xLoopCount 
)

Play Program.

This function opens the chosen file on the SD card, reads it line by line and sends to the ARM COM task. This continues until the end of the file has been reached, at which point the file is reopened and the process repeats until the loop count value has been reached.

Parameters:
[in] pcFileName Contains the file name to play.
[in] xLoopCount Contains the number of times to play the file.

Void.
